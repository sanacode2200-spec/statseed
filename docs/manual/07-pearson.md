# Pearson 相関（2つの数値の直線的な関連）

## この検定はいつ使うか

2つの連続変数が**直線的に**関連しているか（一方が増えると他方も増える／減る）を調べるときに使います。相関係数 r は -1〜+1 の値を取ります。

**たとえば：** 握力と歩行速度に正の相関があるか。

## 操作手順

### 1. データを確認する

CSVを読み込み、解析に使う変数と欠損の状況を確認します。

![Pearson 相関（2つの数値の直線的な関連） データ確認](../../screenshots/manual/pearson/01_data_check.png)

### 2. 検定と変数を選ぶ

「群の差・関連」ページで「CSVから選択」を選びます。

検定の種類で **Pearson 相関** を選びます。

x の列と y の列を指定します。

![Pearson 相関（2つの数値の直線的な関連） 変数選択](../../screenshots/manual/pearson/02_select.png)

### 3. 解析を実行して結果を見る

「検定を実行」を押すと、統計量・p値・95%信頼区間と、日本語の解釈が表示されます。

![Pearson 相関（2つの数値の直線的な関連） 結果](../../screenshots/manual/pearson/03_result.png)

## 結果の読み方

**r** が +1に近いほど強い正の相関、-1に近いほど強い負の相関、0に近いと無相関です。**95%信頼区間**と**p値**もあわせて表示されます。目安として |r|=0.2弱い / 0.4中程度 / 0.7強い。

## よくあるつまずきポイント

- **直線的でない関連は検出できません。** 散布図で形を必ず確認しましょう。
- 外れ値1点で r が大きく動くことがあります。
- 曲線的・順位的な関連や外れ値が気になるときは[Spearman 相関](./08-spearman.md)を使います。
